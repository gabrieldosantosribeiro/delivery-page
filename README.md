## 📖 Sobre o projeto

Este projeto consiste em uma **landing page de delivery**, desenvolvida totalmente do zero utilizando **HTML, CSS e JavaScript**.

O foco foi aplicar na prática conceitos fundamentais de desenvolvimento web, criando uma interface moderna, organizada e funcional.

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

---

## 💡 Funcionalidades

* Layout moderno e organizado
* Navegação entre seções
* Estrutura completa de landing page
* Interface responsiva
* Seções:

  * Início
  * Sobre
  * Serviços
  * Aplicativo
  * Contato
